# 百度语音识别配置和测试

## 🔑 第一步：配置API密钥

### 1. 获取密钥（5分钟）

1. 访问：https://ai.baidu.com/
2. 登录 → 控制台 → 语音技术 → 语音识别
3. 创建应用 → 获取 **API Key** 和 **Secret Key**

### 2. 配置到代码（2分钟）

需要修改**两个文件**：

#### 文件1：BaiduAsrHelper.java（录音文件识别）

```
位置：app/src/main/java/com/cloudink/app/asr/BaiduAsrHelper.java
第18-19行
```

```java
private static final String API_KEY = "你的API_KEY";
private static final String SECRET_KEY = "你的SECRET_KEY";
```

#### 文件2：BaiduRealtimeAsrHelper.java（实时语音识别）

```
位置：app/src/main/java/com/cloudink/app/asr/BaiduRealtimeAsrHelper.java
第28-29行
```

```java
private static final String API_KEY = "你的API_KEY";
private static final String SECRET_KEY = "你的SECRET_KEY";
```

**注意：** 两个文件使用**相同的密钥**

## ❌ 常见错误

### 错误1："请先配置API密钥"
→ 检查是否修改了两个文件的密钥

### 错误2："获取access token失败"
→ 检查密钥是否正确，是否有网络连接

### 错误3："录音初始化失败"
→ 检查是否授予了麦克风权限

### 错误4："未识别到语音内容"
→ 说话声音太小或环境太吵
