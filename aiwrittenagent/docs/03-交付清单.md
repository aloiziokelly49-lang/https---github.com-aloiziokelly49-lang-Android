# 云墨 (CloudInk) — 交付清单（分阶段里程碑与测试检查点）

> **使用方式**: 每完成一个阶段，必须在 **测试检查点** 确认全部通过后，才能进入下一阶段。
> **给开发者的提醒**: 请在每个阶段末尾的 `□ 签字确认` 框手动确认通过（可自行改为 `☑`）。

---

## 阶段总览

```
Phase 0  →  环境与基础设施配置        (1-2 天)
Phase 1  →  认证 + 首页 + 数据库      (3-5 天)
Phase 2  →  OCR + 语音 + 手写引擎     (7-10 天)
Phase 3  →  分屏阅读 + 导出 + 历史     (5-7 天)
Phase 4  →  联调集成 + 性能优化        (3-5 天)
Phase 5  →  交付物准备 + 演示验收      (2-3 天)
```

## 开源代码使用特别提醒

你的项目涉及以下开源代码，**在对应阶段需要额外关注**：

| 开源项目/库 | 涉及阶段 | 关注点 |
|-------------|----------|--------|
| Google ML Kit | Phase 2 | 模型文件自动下载（首次需联网）；许可证 Apache 2.0，无需特殊处理 |
| `saurabhdaware/text-to-handwriting` | Phase 2 | MIT 许可证，允许移植。**若移植代码 → 保留原始版权声明**；若仅参考数学公式 → 标注"inspired by" |
| Room / DataStore / Lifecycle | Phase 0/1 | Apache 2.0，需在 `开源许可证清单.md` 列出 |
| EventBus | Phase 0 | Apache 2.0 |
| CameraX | Phase 2 | Apache 2.0 |
| 自研 Canvas 渲染 | Phase 2 | 如参考多篇开源方案，需在代码注释中标注来源 URL |

**核心原则**:
1. **MIT / Apache 2.0**: 可商用，保留版权声明即可
2. **GPL**: 严禁使用，会导致整个项目强制开源
