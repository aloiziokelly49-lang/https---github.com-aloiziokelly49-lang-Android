## 问题排查

### 如果录音转文字无法插入草稿：

1. **检查权限**
   ```
   设置 → 应用 → 云墨 → 权限 → 麦克风
   ```

2. **检查转写状态**
   - 录音后等待状态提示"转写完成"
   - 确认"实时转写"区域有文字内容

3. **检查按钮状态**
   - "插入草稿"按钮应该是可点击状态
   - 如果按钮灰色，说明转写内容为空

### 如果草稿编辑器无法滚动：

1. **增加文本内容**
   - 输入至少20行文本
   - 确保内容超过输入框高度

2. **正确的滑动方式**
   - 在输入框内部滑动
   - 不要在边缘或外部滑动

3. **获得焦点**
   - 点击输入框获得焦点
   - 光标应该在输入框内闪烁
