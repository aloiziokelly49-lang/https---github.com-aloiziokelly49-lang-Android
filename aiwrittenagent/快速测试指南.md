# 快速测试指南

## 🚀 立即测试修复效果

### 测试1：PDF显示问题（最重要）

**步骤：**
1. 编译并安装应用到vivo X90设备
2. 打开应用，进入"分屏阅读"功能
3. 点击顶部工具栏（标题栏）
4. 选择任意PDF文件
5. **预期结果：** 左侧面板应该显示PDF页面

**如果仍然空白：**
```bash
# 连接设备后运行此命令查看日志
adb logcat | grep -E "PdfViewerAdapter|SplitReaderActivity"
```

**关键日志：**
- ✅ 看到 "PDF 加载成功，页数: X" - PDF文件已打开
- ✅ 看到 "RecyclerView 宽度: XXX" - 宽度已获取
- ✅ 看到 "第 X 页渲染完成" - 页面已渲染
- ❌ 如果看到 "RecyclerView 宽度: 0" - 布局问题
- ❌ 如果看到 "Exception" - 有错误发生

### 测试2：录音转文字功能

**前提条件：** 需要先配置百度API密钥（见下方）

**步骤：**
1. 进入"语音转写"功能
2. 点击"导入录音"按钮
3. 选择一个录音文件（WAV、M4A等格式）
4. 导入成功后，会显示录音信息卡片
5. 点击"识别为文字"按钮
6. 等待几秒钟
7. **预期结果：** 识别的文字会自动添加到文本框中

**如果提示配置API密钥：**
1. 访问 https://ai.baidu.com/
2. 注册并创建应用
3. 获取 API Key 和 Secret Key
4. 打开文件：`app/src/main/java/com/cloudink/app/asr/BaiduAsrHelper.java`
5. 找到这两行：
   ```java
   private static final String API_KEY = "YOUR_API_KEY_HERE";
   private static final String SECRET_KEY = "YOUR_SECRET_KEY_HERE";
   ```
6. 替换为你的实际密钥
7. 重新编译安装

## 🔍 详细调试步骤

### 如果PDF显示空白

**步骤1：查看完整日志**
```bash
adb logcat -c  # 清空日志
adb logcat | grep -E "PdfViewerAdapter|SplitReaderActivity" > pdf_debug.log
```

**步骤2：分析日志**

查找以下关键信息：

1. **PDF是否成功打开？**
   ```
   SplitReaderActivity: PDF 加载成功，页数: 10
   ```

2. **RecyclerView宽度是否正常？**
   ```
   PdfViewerAdapter: RecyclerView 宽度: 648
   ```
   - 如果是0，说明布局有问题
   - 正常应该是几百像素

3. **是否有渲染日志？**
   ```
   PdfViewerAdapter: 第 0 页渲染完成
   ```

4. **是否有错误？**
   ```
   PdfViewerAdapter: 渲染第 0 页时出错
   ```

**步骤3：尝试不同的PDF**

- 尝试小PDF（1-2页）
- 尝试大PDF（50+页）
- 尝试不同来源的PDF

**步骤4：检查布局**

在 `activity_split_reader.xml` 中，RecyclerView的约束：
```xml
<androidx.recyclerview.widget.RecyclerView
    android:id="@+id/rv_pdf_pages"
    android:layout_width="0dp"  <!-- 应该是0dp，使用约束 -->
    android:layout_height="0dp"
    app:layout_constraintStart_toStartOf="parent"
    app:layout_constraintEnd_toStartOf="@id/split_bar"
    ...
/>
```

## 🐛 常见问题快速解决

### Q1: PDF显示空白

**快速检查：**
```bash
adb logcat | grep "RecyclerView 宽度"
```

如果看到宽度为0，尝试：
1. 旋转屏幕（横屏→竖屏）
2. 退出重新进入
3. 重启应用

### Q2: 录音识别失败

**快速检查：**
1. 是否配置了API密钥？
2. 是否有网络连接？
3. 音频格式是否支持？

**测试网络：**
```bash
adb shell ping -c 3 baidu.com
```

### Q3: 输入法语音不工作

**快速检查：**
1. 是否安装了搜狗输入法？
2. 是否设为默认输入法？
3. 是否授予了麦克风权限？
